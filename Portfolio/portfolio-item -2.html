Noughts And Crosses

C++

The basic development of Noughts and Crosses includes different functions which are blocks of code that only runs when called. Functions are important as they perform certain actions.

We have been tasked to create a Noughts and Crosses game by the assignment brief. A development company is looking for a junior developer to showcase their skills and experience with a high-level programming language, C++ has been chosen as the programming language of choice. After successfully building the game, three stages would have been covered, these are designing, developing, and testing. These three steps are important as the task was designed to assess the junior developers understanding of these steps. The assignment brief also states we must include, Pseudo-code, flow charts, function catalogues, test plans as well as interface sketches, as part of the documentation along with the game.. The brief also states that the program (Noughts and Crosses) will include, variables, Boolean operators, conditional statements, loops, functions, libraries, and strings.
